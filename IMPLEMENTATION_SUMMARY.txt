🚀 HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Set GEMINI_API_KEY in backend/.env
2. Start backend: cd backend && npm run dev
3. Start frontend: cd frontend && npm run dev
4. Navigate to Stories page
5. Click "✨ Generate New Story"
6. Choose:
   Option A: Select theme, language, level, etc.
   Option B: Toggle "Use custom prompt" and describe your story
7. Click "Generate Story"
8. Wait 3-10 seconds
9. New story appears in the list!
10. Click to practice
